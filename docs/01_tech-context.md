# Teknoloji Yığını

## Framework
- **Flutter** (cross-platform)

## Tema
- **Sadece koyu tema** (`bg-[#121212]`, `bg-[#111714]`, `bg-[#1A181D]`)
- Renk değişkenleri:
  - `--pastel-green: #A2E4B8`
  - `--pastel-yellow: #F8E4A0`
  - `--pastel-red: #F4A8A8`
  - `--primary-color: #38e07b` (Yeni Ürün Ekle)
  - `--primary-purple: #994ce6` (Yeni Çanta)

## İkonlar
- `Material Symbols Outlined` (Google)
- Kategori ikonları:
  - Su → `water_drop`
  - Yiyecek → `utensils`
  - İlk Yardım → `medical_services`
  - Battaniye → `bed`
  - El feneri → `flashlight_on`
  - Pusula → `explore`
  - Silah/mermi → `military_tech` (isteğe bağlı)

## Veri
- Yerel veritabanı: `Hive`
- Resimler: yerel depolama (`path_provider`)
- API anahtarı yok → tamamen offline

## Bildirimler
- Yerel bildirimler (`flutter_local_notifications`)
- Hatırlatma süresi: kullanıcı tanımlı (örneğin 7 gün önce)
